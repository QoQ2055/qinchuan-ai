# 秦川线上简历设计

## 目标

将《秦川-动画设计》PDF 转为一个私密、可分享的单页线上简历，面向动画设计与 AI 动画导演/制片相关招聘场景。

## 视觉方向

采用已确认的深色电影感：墨黑背景、暖白文字与酸性黄点缀；采用编辑式网格、大字号姓名和细线信息分隔。视觉重心是“AI 动画导演 / 制片人”定位，而非传统表格简历。

## 页面结构

1. 首屏：姓名、角色、北京、11 年经验、联系方式与核心定位。
2. 个人优势：AI 创作与传统设计能力、智能体工作流、制作提效说明。
3. 精选经历：突出北京看云控股、福建泽联、杭州蔻艺及设计管理经验，完整保留其它履历作为可展开时间线。
4. 项目经验、教育背景和资格证书。
5. 页尾联系区，复制或点击邮箱/电话即可联系。

## 交互与响应式

单页锚点导航；手机端将双栏布局折叠为单列。没有登录、表单或数据存储。邮件与电话使用原简历数据，部署使用私密链接。

## 验证

构建必须成功；检查页面元数据、桌面与移动端排版、链接可用性，并发布最终站点。
