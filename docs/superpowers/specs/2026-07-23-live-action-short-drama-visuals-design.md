# 仿真人短剧视觉档案与轻量动效规格

## 目标

为“AI仿真人短剧开发”补足导演作品集的影像表达：用三张写实、但不冒充实际成片的视觉概念主镜，连同一张制作档案图，建立从叙事气质到制片流程的可读结构；为页面补充克制的互动反馈。

## 内容与命名边界

- 生成图均为 **Visual Concept Stills / 仿真人短剧视觉概念帧**，不是《撕掉婚纱后》《重生十八岁我拿下了进修名额》《垫付两万八，我拒了亿级大单》的真实剧照、海报或官方物料。
- 图片不含可读文字、剧名、人物姓名、商标或水印；三部代表作仍由现有 HTML 文本列出。
- 后续如提供获授权的真实剧照，可在不改变布局和动效接口的前提下替换任一概念帧。

## 视觉与版式

- 在“AI仿真人短剧开发”项目卡之后插入 `short-drama-reel` 区块，使用“制作档案 / 03”索引和短说明，明确其为视觉概念帧。
- **三张主镜**为同一套低饱和、写实电影静帧，桌面三列等宽接触印样，手机单列。每张采用 4:5 纵向比例，题材依序为：婚礼反转、重返人生的决定时刻、都市关系中的艰难抉择。
- **一张制作档案图**置于三张主镜下方，采用宽幅 16:9，画面为剪辑台、监看屏、镜头清单和时间轴；它只承担制作流程气氛，不写入虚构项目事实。
- 三张概念帧整体偏黑灰、暖象牙肤色与环境光，局部使用现有荧光黄作为极少量实景光源／信号色；不使用夸张海报红、赛博霓虹或人物特写拼贴。
- 容器沿用现有细灰线和黑色底，不增加新的粗线框或长装饰线。

## 动效

- 支持动画的设备上，区块进入视口后一次性淡入并上移不超过 12px；三张主镜以 80ms 的微小错峰出现。
- 悬停时，主镜图只缓慢放大至最多 `1.02`；图上叠加非事实性的镜头编号（`CONCEPT FRAME / 01–03`），并提高边框亮度。
- 现有项目卡维持只改变边框／背景亮度的规则，不添加位移；制作档案图只允许现有的 1–2% 微缩放反馈。
- `prefers-reduced-motion: reduce` 下，所有新增淡入、错峰、缩放、过渡与编号显隐均立即静止；触屏设备不依赖悬停。

## 可访问性与验证

- 每张图使用准确的中文 `alt`，描述画面类型而不把它描述为真实作品剧照。
- 宽度不超过既有项目内容列；窄屏下主镜与文字均为一列，页面不得产生横向溢出。
- 渲染测试覆盖“视觉概念帧”标记、四张图片路径、四个 `alt`、三列到单列 CSS 规则和减少动态效果规则。
- 完整构建、GitHub Pages 静态导出和公开链接检查必须验证四个图片在 `/qinchuan-ai/images/` 下可加载。
